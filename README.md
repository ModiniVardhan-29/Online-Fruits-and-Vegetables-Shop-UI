#  FreshMart – Online Fruits and Vegetables Shop UI

##  Project Overview

FreshMart is a responsive and interactive front-end web application designed for online fruits and vegetables shopping. The project was developed as part of the **User Interface Design (UID)** course to demonstrate modern web design principles, intuitive navigation, and user-friendly shopping experiences.

The application allows users to browse fresh fruits and vegetables, search for products, view product details, manage a shopping cart, and proceed to checkout through an attractive and responsive interface.

---

##  Problem Statement

Many online grocery platforms have complicated navigation structures, cluttered layouts, and poor user experiences that make it difficult for customers to find products quickly.

FreshMart addresses these challenges by providing:

* A clean and intuitive user interface
* Fast product search functionality
* Easy product browsing and categorization
* Interactive shopping cart management
* Responsive design for multiple devices

---
